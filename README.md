Description
--

This depository contains the code for the article [Chemical Reaction Networks Learn Better than Spiking Neural Networks](https://arxiv.org/abs/2603.12060). 

Usage
--
The file 'CRN.ipynb' contains the code to simulate the CRN and train it on the handwritten digits dataset of scikit-learn. It enables to reproduce Figures 3 and 4.
